---
title: Your First Conversation
description: Talk to the agent, run a skill, and approve an action.
---

With a model configured, open the sidebar (**`Ctrl+Shift+A`** / **`Cmd+Shift+A`**)
and try a few things.

## Ask about the current page

Type a request in the input box and send it. The agent can see the page you are
viewing, so you can ask things like *"summarize this article"* or *"what are the
key dates on this page?"*

## Let it act for you

Ask the agent to *do* something — *"open the docs and find the install command"* or
*"fill this form with my details"*. The agent will use its
[browser tools](/docs/features/browser-automation) to carry it out.

By default the agent **asks for confirmation** before taking actions. You will see a
permission prompt; approve or deny it. You can change this to auto-execute under
[Settings → General](/docs/settings/general).

## Run a skill

Type a slash command to invoke a [skill](/docs/skills) directly, for example:

```text
/app a calculator with a dark theme
```

```text
/brain what do I know about project Acme?
```

If you do not use a slash command, the agent will load a matching skill on its own
when your request fits one.

## Where conversations go

Open **`nevoflux://history`** (or the history panel in the sidebar) to revisit past
sessions.
