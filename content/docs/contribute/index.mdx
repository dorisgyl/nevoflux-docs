title: Contributing
description: How to contribute to NevoFlux and its documentation.
---

Contributions are welcome — code, docs, translations, and bug reports.

## Ways to help

- File issues and feature requests on [GitHub](https://github.com/dorisgyl/nevoflux).
- Improve these docs (every page links to its source).
- Help translate the docs into more languages.

## Editing the docs

Documentation lives in `content/docs` as MDX. English files are `*.mdx`;
Simplified Chinese files use the `*.zh.mdx` suffix.
