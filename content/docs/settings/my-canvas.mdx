---
title: My Canvas
description: Browse and manage the Canvas artifacts you've created or imported.
---

The **My Canvas** tab lists the [Canvas](/en/docs/canvas) artifacts saved on your
machine — apps, documents, diagrams, and more that the agent built for you or that
you imported.

## Find and open

- **Search** by name.
- **Type filter** — All / HTML / Markdown / SVG / JSON / Project.
- **Source filter** — All / From sessions / Imported.

Open an artifact to load it (`nevoflux://canvas/{id}`), or delete ones you no longer
need.

*Example:* set **Source** to **Imported** and **Type** to **HTML** to find a canvas a
colleague shared with you via a `nevoflux://import/{share_id}` link, then open it.

See [Canvas](/en/docs/canvas) for how artifacts are created, edited, exported, and
shared.
