The **Shortcuts** tab is a read-only reference of NevoFlux's keyboard shortcuts.

| Action | Windows / Linux | macOS |
| --- | --- | --- |
| Show / hide the AI agent sidebar | **`Ctrl+Shift+A`** | **`Cmd+Shift+A`** |

> This is the shortcut to remember: **`Ctrl+Shift+A`** (or **`Cmd+Shift+A`** on
> macOS) toggles the agent sidebar from anywhere in the browser.
