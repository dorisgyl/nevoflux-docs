**My Canvas** 标签页列出保存在你机器上的 [Canvas](/zh/docs/canvas) 制品 —— 智能体
为你构建的、或你导入的应用、文档、图示等。

## 查找与打开

- 按名称 **Search(搜索)**。
- **Type filter(类型筛选)** —— All / HTML / Markdown / SVG / JSON / Project。
- **Source filter(来源筛选)** —— All / From sessions(来自会话) / Imported(导入)。

打开制品即可加载它(`nevoflux://canvas/{id}`),也可删除不再需要的制品。

*示例:* 把 **Source** 设为 **Imported**、**Type** 设为 **HTML**,即可找到同事通过
`nevoflux://import/{share_id}` 链接分享给你的 canvas,然后打开它。

制品如何创建、编辑、导出与分享,见 [Canvas](/zh/docs/canvas)。
