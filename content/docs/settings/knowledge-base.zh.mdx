**Knowledge Base** 标签页用于安装和管理 **GBrain** —— NevoFlux 的本地知识库。它会
替你运行安装向导;所有数据都不离开你的机器。

## 状态

该标签页显示安装/设置 **Status** 徽标和实时 **Server** 徽标,以及详情:已安装的
**bun** 版本、**gbrain** 版本,以及你的 **brain dir** 路径。

## 启用

点击 **Enable Knowledge Base** 运行向导,它会:

1. 安装 **bun**(GBrain 使用的 JS 运行时)。
2. 安装 **gbrain** 工具。
3. 初始化你的大脑(本地向量 + markdown 存储)。
4. 启动知识库服务。

进度会在对话框中实时显示。当 **Server** 徽标显示 *ready* 时,GBrain 已启用。

## 管理

- **Refresh** —— 重新检查状态。
- **Restart server** —— 若服务停止则恢复。
- **Update gbrain…** —— 更新到更新版本的 gbrain。
- **Browse Knowledge Base →** —— 打开 `nevoflux://brain` 阅读你的页面。

*示例:* 在全新安装中打开此标签页,点击 **Enable Knowledge Base**,等待 *ready*,
然后点击 **Browse Knowledge Base →**。此后在阅读文章时,让智能体执行
`/brain-capture 保存这个页面`。

如何捕获、回忆、组织知识,以及数据存放位置,见
**[知识库(GBrain)](/zh/docs/knowledge-base)**。
