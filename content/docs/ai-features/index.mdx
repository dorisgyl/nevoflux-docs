NevoFlux is built around an AI agent that can read pages, take actions, and
automate multi-step tasks on your behalf.

## Highlights

- **Command** — describe a task in natural language and let the agent execute it.
- **Context** — the agent understands the page you are looking at.
- **Automation** — chain steps across sites into repeatable flows.
